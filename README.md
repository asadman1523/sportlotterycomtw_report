# 運彩視覺化UI

這是一款專為台灣運彩（Taiwan Sports Lottery）設計的 Chrome 瀏覽器擴充功能。它能在「我的投注」頁面中，自動攔截並彙整您所設定的歷史注單資料，並以清晰的視覺化介面展示您的資金去向與戰績損益。

## 主要功能

- 免費版可查詢最近一年內注單，並提供 24 小時、7 天、30 天快捷查詢。
- Pro 買斷版可查詢最近二年內注單，並解鎖 1 小時、3 小時、6 小時、12 小時快捷查詢。
- 可依球類分類篩選注單，例如足球、籃球、棒球、網球與其他分類。
- 分別顯示「總投入」、「未派彩本金」、「已結算本金」、「總派彩」與「淨損益」。
- 支援將報表匯出成 CSV 檔案，方便使用 Excel 等工具做進一步的分析。

## 📥 安裝說明

1. 從 Chrome Web Store 安裝：[運彩視覺化UI](https://chromewebstore.google.com/detail/idjjhknkieldnadadngeogjiiiegiihj?hl=zh-TW)。
2. 如需手動安裝，可從 GitHub 下載此專案的 ZIP 檔並解壓縮，或是直接 `git clone` 到您的電腦中。
3. 打開 Chrome 瀏覽器，進入擴充功能管理頁面（在網址列輸入 `chrome://extensions/`）。
4. 在右上角開啟 **「開發人員模式」** (Developer mode)。
5. 點擊左上角的 **「載入未封裝項目」** (Load unpacked)。
6. 選擇您剛剛解壓縮出來的資料夾。
7. 完成安裝！

## 💡 使用方式

1. 安裝完成後，請登入 [台灣運彩官方網站](https://www.sportslottery.com.tw/)。
2. 點擊右上角進入 **「我的帳戶」 -> 「我的投注」** 頁面。
3. 擴充功能將會自動在背景啟動，一旦偵測到注單資料載入，就會在畫面下方彈出**運彩視覺化UI**！
4. **搜尋特定日期**：可直接在外掛 UI 中設定日期區間並按下搜尋，也可以使用 24 小時、7 天、30 天快捷按鈕，報表會依照指定區間重新載入。
5. **篩選球類分類**：可點選足球、籃球、棒球、網球或其他分類，快速篩選目前報表中的注單。

## Pro 買斷解鎖

- 免費版最多查詢最近一年；超過一年或點擊 1 小時、3 小時、6 小時、12 小時快捷按鈕時，會顯示 Pro 提示。
- Pro 為買斷制，啟用後最多可查詢最近二年，並解鎖 1/3/6/12 小時快捷查詢。
- 購買請透過 Line 官方帳號：[https://lin.ee/zsGJ9oT](https://lin.ee/zsGJ9oT)。
- Pro 提示會顯示可複製的 Base64 裝置碼資料。付款時請提供該資料，確認後會收到一組綁定 Chrome Google 帳號的啟動序號。
- 啟動序號會儲存在本機與 Chrome 同步儲存空間；重裝外掛後若 Chrome 同步資料可用，登入同一個 Chrome Google 帳號即可自動恢復 Pro，否則可用同一組啟動序號重新啟用。
- 若尚未登入 Chrome，Pro 提示會顯示「請先登入 Chrome」。
- Pro 只解鎖查詢便利性，不提供下注建議、自動投注、代操、帳密同步或 OTP 相關功能。實際可查資料仍以台灣運彩官方網站可提供的資料為準。
