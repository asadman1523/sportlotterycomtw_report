# 運彩視覺化UI

這是一款專為台灣運彩（Taiwan Sports Lottery）設計的 Chrome 瀏覽器擴充功能。它能在「我的投注」頁面中，自動攔截並彙整您所設定的歷史注單資料，並以清晰的視覺化介面展示您的資金去向與戰績損益。

## 🌟 主要功能

- **自動化資金彙整**：一眼看清「總投入」、「未派彩本金」、「已結算本金」、「總派彩」與「淨損益」的數學關係。
- **無縫攔截 API**：自動攔截官方網頁的網路請求，100% 準確抓取資料。
- **與官方日期過濾器連動**：不需額外設定，只要在官網的「我的投注」頁面選擇您想要的日期區間並按下搜尋，報表就會自動更新為該區間的統計結果！
- **欄位排序功能**：支援對「下注時間」、「玩法」、「投注額」、「預計/實際派彩」及「狀態」進行升冪或降冪排序。
- **一鍵匯出 CSV**：支援將報表匯出成 CSV 檔案，方便使用 Excel 等工具做進一步的分析。
- **自訂開啟偏好**：首次安裝預設自動彈出報表，您可以透過勾選框自由切換下次載入時是否要預設開啟。

## 📥 安裝說明

1. 從 GitHub 下載此專案的 ZIP 檔並解壓縮，或是直接 `git clone` 到您的電腦中。
2. 打開 Chrome 瀏覽器，進入擴充功能管理頁面（在網址列輸入 `chrome://extensions/`）。
3. 在右上角開啟 **「開發人員模式」** (Developer mode)。
4. 點擊左上角的 **「載入未封裝項目」** (Load unpacked)。
5. 選擇您剛剛解壓縮出來的資料夾。
6. 完成安裝！

## 💡 使用方式

1. 安裝完成後，請登入 [台灣運彩官方網站](https://www.sportslottery.com.tw/)。
2. 點擊右上角進入 **「我的帳戶」 -> 「我的投注」** 頁面。
3. 擴充功能將會自動在背景啟動，一旦偵測到注單資料載入，就會在畫面下方彈出**運彩視覺化UI**！
4. **過濾特定日期**：請直接使用運彩網頁上的「日期搜尋」功能，設定好日期後按下搜尋，擴充套件就會立刻套用該日期區間並重新產生報表！
